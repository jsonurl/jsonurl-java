The jsonurl-core package defines provides classes to parse and generate JSON→URL text.

The Streaming API

The JsonUrlIterator interface defines an API for parsing JSON→URL text as a sequence of events. It may be used directly, but more often one of the object model APIs (e.g. org.json.j2se, org.json.jsonorg, org.json.jsonp) is used.

Security

The parser is designed to parse untrusted input. It supports limits on the number of parsed values and depth of nested arrays or objects. When the limit is exceeded a LimitException is thrown. Sane limit values are set by default.
