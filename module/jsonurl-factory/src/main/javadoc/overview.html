The jsonurl-factory package provides an extensible JSON→URL parser core and an implementation based on Java SE datatypes.

Usage

The Parser interface defines a generic JSON→URL object model parser and JsonUrlParser provides an implementation based Java SE data types (e.g. Map, List, etc). There are additional modules, distributed as separate artifacts, which implement JSON→URL parsers based on other APIs (json.org API, JSR-374 API).

Here's a simple example of parsing JSON→URL text into a Java Map. 

import java.util.Map;
import org.jsonurl.j2se.JsonUrlParser;

JsonUrlParser p = new JsonUrlParser();
Map myObject = p.parseObject( "(Hello:World)" );

Security

The parser is designed to parse untrusted input. It supports limits on the number of parsed values and depth of nested arrays or objects. When the limit is exceeded a LimitException is thrown. Reasonable limit values are set by default.
