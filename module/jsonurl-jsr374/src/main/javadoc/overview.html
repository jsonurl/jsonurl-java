The jsonurl-jsr374 package provides a JSON→URL parser bound to JSR-374 types.

Usage

The JsonUrlParser class may be used to parse JSON→URL text to build instances of JsonObject and/or JsonArray. 

import javax.json.JsonObject;
import org.jsonurl.jsonp.JsonUrlParser;

JsonUrlParser p = new JsonUrlParser();
JsonObject myObject = p.parseObject( "(Hello:World)" );

Security

The parser is designed to parse untrusted input. It supports limits on the number of parsed values and depth of nested arrays or objects. When the limit is exceeded a LimitException is thrown. Reasonable limit values are set by default.
